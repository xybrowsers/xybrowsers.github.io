# 隐私政策 / Privacy Policy

**最后更新日期 / Last Updated**: 2026-02-02

## 中文版

### 引言

欢迎使用我们的应用程序（以下简称"本应用"）。我们深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。我们致力于维持您对我们的信任，恪守以下原则保护您的个人信息：权责一致原则、目的明确原则、选择同意原则、最少够用原则、确保安全原则、主体参与原则、公开透明原则等。

本隐私政策旨在向您说明我们如何收集、使用、存储、共享和保护您的个人信息，以及您可以如何管理您的个人信息。

### 一、我们如何收集和使用您的个人信息

我们仅会出于本政策所述的以下目的收集和使用您的个人信息：

#### 1.1 为您提供浏览服务

**基本浏览功能**
- **网页访问记录**：为了提供浏览历史记录功能，我们会在本地存储您访问过的网页URL、标题、访问时间等信息
- **书签数据**：为了提供书签管理功能，我们会在本地存储您收藏的网页URL、标题等信息
- **Cookie和缓存**：为了提升网页加载速度和用户体验，我们会在本地存储网页Cookie和缓存数据

#### 1.2 为您提供搜索服务

**搜索功能**
- **搜索关键词**：当您使用搜索功能时，我们会将您输入的关键词发送给您选择的搜索引擎（百度、谷歌或必应）
- 注意：搜索引擎的数据使用遵循其各自的隐私政策

#### 1.3 为您提供扩展功能

**二维码扫描**
- **相机权限**：当您使用二维码扫描功能时，我们需要访问您的相机
- **用途说明**：仅用于扫描二维码并识别其中的URL，不会保存或上传任何图像数据

**网页截图**
- **截图数据**：当您使用网页截图功能时，我们会生成当前网页的截图
- **相册权限**：保存截图时需要访问您的相册权限
- **用途说明**：截图仅保存在您的本地设备，不会上传到任何服务器

**麦克风权限**（如适用）
- **语音输入**：如果您使用语音输入功能，我们需要访问麦克风权限
- **用途说明**：语音数据仅用于转换为文本，不会被保存或上传

### 二、我们如何存储您的个人信息

#### 2.1 存储地点

您的个人信息将被存储在您的设备本地。我们不会将您的个人浏览数据上传到任何远程服务器。

#### 2.2 存储期限

除非您主动删除，否则我们会持续在本地保存您的浏览历史、书签等数据。您可以随时通过应用内的清除功能删除这些数据。

#### 2.3 存储安全

我们采用业界标准的安全技术措施保护您存储在设备上的数据，包括：
- 数据加密存储
- 安全的密钥管理
- 定期的安全审查

### 三、我们如何共享、转让、公开披露您的个人信息

#### 3.1 共享

我们不会与任何第三方公司、组织和个人共享您的个人信息，但以下情况除外：
- 在获取明确同意的情况下：获得您的明确同意后，我们会与其他方共享您的个人信息
- 在法律法规规定或监管要求的情况下

#### 3.2 转让

我们不会将您的个人信息转让给任何公司、组织和个人，但以下情况除外：
- 在获取明确同意的情况下
- 在涉及合并、收购或破产清算时，如涉及到个人信息转让，我们会要求新的持有您个人信息的公司、组织继续受本隐私政策的约束

#### 3.3 公开披露

我们不会公开披露您的个人信息，除非：
- 获得您的明确同意
- 基于法律法规、法律程序、诉讼或政府主管部门强制性要求

### 四、您如何管理您的个人信息

#### 4.1 访问和管理您的信息

您可以通过以下方式访问和管理您的信息：
- **浏览历史**：在应用内的历史记录页面查看和删除
- **书签**：在应用内的书签页面查看和管理
- **设置**：在设置页面管理应用权限和偏好设置

#### 4.2 删除您的信息

您可以通过以下方式删除您的信息：
- 单独删除特定的历史记录或书签
- 在设置中清除所有浏览数据
- 卸载应用将删除所有本地存储的数据

#### 4.3 撤回同意

您可以通过以下方式撤回您的授权同意：
- 在设备系统设置中关闭相应权限（相机、相册、麦克风等）
- 关闭权限可能会导致部分功能无法使用

### 五、第三方服务

本应用可能包含指向第三方网站或服务的链接。当您点击这些链接时，您将离开本应用并访问第三方服务。这些第三方服务有自己的隐私政策，我们建议您仔细阅读。

特别说明：
- **搜索引擎**：您选择的搜索引擎（百度、谷歌、必应）有各自的隐私政策
- **访问的网站**：您通过本应用访问的所有网站都有各自的隐私政策

### 六、未成年人保护

我们非常重视对未成年人个人信息的保护。如果您是未满18周岁的未成年人，请在监护人的陪同下阅读本政策，并在使用我们的服务前确保已取得监护人的同意。

### 七、本隐私政策如何更新

我们可能会不时更新本隐私政策。当隐私政策发生变更时，我们会在应用内通过弹窗、公告等形式向您展示变更后的隐私政策。
